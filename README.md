## How To

### Run Project

1.  Clone Project
    ```
    git clone https://github.com/pspapleng/NestJS-todo.git
    npm install
    ```
2.  Run Database As Container.
    ```
    docker-compose up -d
    ```
3.  Run Project.
    ```
    npm run start:dev
    ```
